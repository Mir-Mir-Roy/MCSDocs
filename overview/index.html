General information

  • This is WORKABLE addon but WITHOUT poka-yoke!
  • Imported camera's and image's names MUST be same accordingly. For example if background image has name "camera_001.jpg", camera name in scene must be the same but without extension - "camera_001".

UI Review

  1. MCS list.
  2. button adds selected cameras into MCS list, - deletes selected camera in MCS list camera.
  3. Check box is to load images from disk always. Useful to update images if they changed in folder.
  4. String field and button to set image folder path.
  5. Add all cameras in scene to MCS list
  6. Select camera in scene highlighted in MCS list.
  7. Clear MCS list.
  8. Indicates name of current active camera in viewport. When pressed - select the camera in scene.
  9. Enable view navigation within the camera view.
  10. Set previous camera in MCS list to viewport.
  11. Set next camera in MCS list to viewport.
  12. Set selected in MCS list camera to viewport.
  13. Delete current active camera in viewport from MCS list.
  14. Adjust near clipping camera distance(in meters).
  15. Adjust far clipping camera distance(in meters).
  16. Set the clipping camera distances.
  17. Fix fbx camera import wrong direction. Select cameras in scene and press the button.

Quick instruction

  • Import fbx/alembic file with cameras into Blender scene.
  • Check the camera names and background images names in folder. They must be the same.
  • Click on MCS tab in N button panel.
  • Press "Add All Scene Cameras to List" button. It's load all cameras into MCS camera list.
  • Press button to select folder with background images. DO not select files in the folder.
  • Just start pressing or buttons to switch cameras in viewport. Background images will load automatically.
  • To assign shortcuts for Prev Cam and Nest Cam buttons use 'mcs_addon.prev_in_list' and 'mcs_addon.next_in_list' addon's operators
